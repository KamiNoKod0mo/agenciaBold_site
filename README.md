# AgenciaBold
Uma aplicação web criada com React.

## Tecnologias
- React 18
- react-slick

## Requisitos
- Node.js v16+ e npm ou Yarn.

## Como executar
1. Clone este repositório:
   ```bash
   git clone https://github.com/KamiNoKod0mo/agenciaBold_site/
   cd agenciaBold_site
   npm install
   npm start
   ```
